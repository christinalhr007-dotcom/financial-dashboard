# 理财师团队管理数据看板

一个基于 Streamlit 的理财师团队管理 Web 数据看板，支持数据上传、可视化展示、人员预警和多周期对比。

## 功能特性

### 📊 数据上传模块
- 支持上传3个Excel文件：
  - 客户触达统计
  - 出单业绩
  - 商机线索汇总
- 自动解析并存储，支持历史数据累积

### 📈 可视化看板
- **团队总览**: 关键指标汇总（覆盖率、出单金额、商机数等）
- **个人详情**: 每个理财师的过程管理+业绩数据
- **趋势图表**: 日度/周度/月度趋势线
- **排名榜单**: 按各维度排名

### ⏰ 时间筛选
- 周度/月度/季度切换
- 环比对比（本期 vs 上期）
- 可选择具体时间范围查看

### ⚠️ 人员预警系统
- 触达覆盖率 < 80% 预警 🔴
- 高质量触达占比 < 50% 预警 🟡
- 无出单人员预警 🟠
- 商机数量过少预警 🟡
- 排名下降超过3名预警 🟠

## 使用方法

### 方式一：上传文件
1. 在左侧边栏选择「上传文件」模式
2. 分别上传三个Excel文件：
   - 客户触达统计（by全球主RM）
   - 出单业绩
   - 广州ARK-LTC商机线索汇总
3. 点击上传后自动解析并展示看板

### 方式二：加载示例数据
1. 在左侧边栏选择「加载示例数据」模式
2. 点击即可加载预置的示例数据进行演示

## 数据格式说明

### 客户触达统计
| 字段 | 说明 |
|------|------|
| 主AR大区 | 大区名称 |
| 主AR片区 | 片区名称 |
| 主AR姓名 | 理财师姓名 |
| 客户数 | 负责客户数量 |
| 综合覆盖率 | 触达覆盖率百分比 |
| 高质量触达 | 高质量触达客户数 |
| 高质量客户占比 | 高质量触达占比 |
| ... | ... |

### 出单业绩
| 字段 | 说明 |
|------|------|
| 日期时间 | 订单日期时间 |
| 业绩员工 | 理财师姓名 |

| 订单营销业绩人民币 | 营销业绩（人民币） |

| 字段 | 说明 |

| 预计投资原币金额/万 | 预计投资金额 |
| 商机状态 | 进行中/已成交/已放弃 |

## 看板模块说明

### 团队总览
- KPI指标卡片：总客户数、平均覆盖率、总出单金额、商机总数
- 本期vs上期环比对比
- 触达覆盖率排名图
- 高质量触达占比排名图

### 个人详情
- 选择理财师查看详细数据
- 触达指标：客户数、覆盖率、服务记录、商机、打款
- 业绩数据：出单金额、营销业绩、出单笔数
- 商机数据：商机总数、预计投资、转化金额

### 趋势分析
- 每日出单金额趋势
- 每日出单笔数趋势
- 产品大类分布饼图
- 商机状态分布饼图

### 排名榜单
- 覆盖率TOP10
- 出单金额TOP10
- 商机数量TOP10
- 营销业绩TOP10

### 预警中心
- 按严重程度分级展示预警
- 🔴 严重预警：覆盖率<60%
- 🟠 中度预警：覆盖率<80%或无出单
- 🟡 轻度预警：高质量占比<50%或商机过少

## 技术栈

- **前端框架**: Streamlit
- **图表库**: Plotly
- **数据处理**: Pandas
- **数据存储**: JSON文件

## 注意事项

1. 首次使用建议先上传完整数据进行初始化
2. 每次上传新数据会自动保存到历史记录
3. 历史数据用于趋势分析和环比对比
4. 预警阈值可在 `alert_system.py` 中自定义调整
